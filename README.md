## üéØ Objectives
- Build an enriched dataset with temporal, calendar, and market variables.  
- Train an advanced ML model (XGBoost / HistGradientBoostingRegressor).  
- Deploy an interactive app to simulate discount scenarios.  
- Generate daily forecasts for November 2025, including Black Friday.  

---

## üõ†Ô∏è Data Science Methodology
The project follows a professional DS/ML cycle:  
1. Problem definition ‚Üí Sales forecasting  
2. Data collection & preparation ‚Üí Historical sales, own & competitor prices, calendar events  
3. Exploratory Data Analysis (EDA) ‚Üí Visualization of patterns by day, category, subcategory, product  
4. Feature Engineering ‚Üí Calendar variables, holidays, Black Friday, Cyber Monday, lags, moving averages  
5. Modeling ‚Üí Training with XGBoost / HistGradientBoostingRegressor  
6. Evaluation ‚Üí Metrics (MAE, MSE, RMSE)  
7. Implementation ‚Üí Streamlit app for simulation and predictions
